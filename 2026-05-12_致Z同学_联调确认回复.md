## 三、关于 `load_sequence()` 的调用方式

你的理解完全正确。`meta_row` 就是直接从 `metadata.csv` 中读取对应行的 **pandas Series**：

```python
import pandas as pd
from data_pipeline_v2 import load_sequence

# 正确用法 ✅
metadata = pd.read_csv("data/metadata.csv")
metadata['filename'] = metadata['filename'].str.strip()

# 找到你要的那个文件
row = metadata[metadata['filename'] == '你要的文件名.csv'].iloc[0]

# 直接传这个 Series 作为 meta_row
x, y, labels = load_sequence("data/dataset/dataset/test/你要的文件名.csv", row, seq_len=200)
# x:       [200, 17]  输入特征
# y:       [200, 2]   标签（已归一化）
# labels:  [200]      anomaly_code

# 然后转 Tensor 传给 predict：
import torch
x_tensor = torch.from_numpy(x).float().unsqueeze(0)  # [1, 200, 17]
thrust, mfr, isp = predict(model, x_tensor)
```

`load_sequence()` 内部会用 `meta_row['test_pressure']`、`meta_row['cumulated_throughput']` 等字段来构建 17 维特征，所以 **metadata 的那一行必须包含完整的字段**。
