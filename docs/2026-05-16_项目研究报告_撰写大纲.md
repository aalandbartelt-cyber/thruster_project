### 4.3 模型架构（约1000字）

```
4.3.1 Attention-LSTM 双输出架构
  - LSTM backbone（2层，256 hidden）
  - Multi-head Self-Attention（4 heads）+ 残差连接 + LayerNorm
  - 双输出头：thrust（线性 + THRUST_SCALE=8.0反归一化）
              mfr（线性 + MFR_MAX=2000.0反归一化）
  - Isp 推导：Isp = F / (ṁ × g₀)

4.3.2 加权损失函数
  Loss = 0.7 × MSE(thrust) + 0.3 × MSE(mfr)
  权重选择依据：thrust 是主要监测指标，mfr 辅助诊断

4.3.3 训练配置
  - 窗口长度 200 步（2秒 @ 100Hz）
  - Batch size 64, Adam (lr=0.001, wd=2e-5)
  - ReduceLROnPlateau (factor=0.5, patience=5)
  - Early stopping (patience=20)

4.3.4 与传统ML的对比论证
  - 表：LSTM vs XGBoost vs LightGBM（引用三项优化实验报告 实验一）
  - LSTM 全11种工况碾压，MFR 逐点 49.1 vs 154.5 mg/s（3.1×差距）
  - 论证：时序记忆是刚需，树模型无法获取跨帧物理信息
```
