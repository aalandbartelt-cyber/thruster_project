## 二、`inference_utils.py` API（供 app_v2.py 调用）

```python
from inference_utils import (
    load_dual_model, predict, predict_with_uncertainty,
    compute_residuals, generate_health_report
)
```

### 加载模型

```python
model = load_dual_model("outputs/models/v2/dual_output_lstm_v2.pth")
# 返回 eval 模式的 model，自动选择 CUDA/CPU
```

### 单次推理（用于实时监控面板）

```python
x_tensor = torch.randn(1, 200, 17)  # [batch, 200步, 17维]
thrust, mfr, isp = predict(model, x_tensor)
# thrust: np.ndarray [1, 200]  单位 N
# mfr:    np.ndarray [1, 200]  单位 mg/s
# isp:    np.ndarray [1, 200]  单位 s

# 支持单个样本（自动 squeeze）
thrust, mfr, isp = predict(model, x_tensor[0])  # 输入[200,17] → 输出[200]
```

### MC Dropout 不确定性量化（置信区间显示）

```python
t_mean, m_mean, i_mean, t_std, m_std, i_std = \
    predict_with_uncertainty(model, x_tensor, n_samples=30)
# t_mean = thrust 均值, t_std = thrust 标准差
# Streamlit 中画置信区间：plt.fill_between(..., t_mean-2*t_std, t_mean+2*t_std)
```

### 残差 + 异常检测

```python
residuals, is_anomaly, anomaly_ratio = \
    compute_residuals(pred_thrust, actual_thrust, threshold=0.6)
# residuals: |pred - actual|
# is_anomaly: bool 数组（残差 > 0.6N 为异常）
# 可用于大屏标注红色告警区间
```

### 一键健康报告

```python
report = generate_health_report(
    thrust, mfr, isp,
    is_anomaly=is_anomaly,
    thrust_good=1.0,  # 推力健康下限
    isp_good=150.0     # 比冲健康下限
)
# report 字典内容：
#   summary:        "推进器健康评分 70/100 —— 正常"
#   thrust_status:  {'value':'1.23 N',  'score':80, 'level':'normal'}
#   mfr_status:     {'value':'512.3 mg/s','score':74, 'level':'warning'}
#   isp_status:     {'value':'165.2 s', 'score':85, 'level':'normal'}
#   anomaly_status: {'anomaly_ratio':0.03, 'n_anomaly':6, ...}
#   overall_health: 70
```

## 四、模型权重（用于 model_onnx_export.py）

```python
import torch
from model_train_v2 import DualOutputLSTM

model = DualOutputLSTM()
model.load_state_dict(torch.load(
    "outputs/models/v2/dual_output_lstm_v2.pth",
    map_location='cpu', weights_only=True
))
model.eval()

# ONNX 导出
dummy = torch.randn(1, 200, 17)
torch.onnx.export(model, dummy, "dual_output_lstm.onnx",
    input_names=['input'], output_names=['output'],
    dynamic_axes={'input':{0:'batch'}, 'output':{0:'batch'}})
```
