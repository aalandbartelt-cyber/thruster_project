## 三、环境搭建

### 3.1 注册华为云并领取免费昇腾算力

1. 打开华为云官网：https://www.huaweicloud.com
2. 手机号注册 → 实名认证（个人即可，无需企业）
3. 领取**200 元无门槛昇腾代金券**（控制台 → 费用中心 → 代金券管理）
   - 足够跑 25 小时 ai2.xlarge.1（32GB 显存），完全够用

### 3.2 创建 SSH 密钥对

1. 控制台搜索「弹性云服务器 ECS」→ 左侧「密钥对」
2. 创建密钥对，名称 `ascend-key`
3. 下载私钥 `ascend-key.pem` → **妥善保存**
4. Windows 用户用 PuTTYgen 转 PPK 格式

### 3.3 创建昇腾云服务器

购买弹性云服务器，配置如下：

| 配置项 | 值 |
|--------|-----|
| 计费模式 | **按需计费**（不用就关机） |
| 区域 | **华北-北京四**（昇腾资源最多） |
| CPU 架构 | **鲲鹏计算**（昇腾必须配鲲鹏） |
| 规格 | **ai2.xlarge.1**（32GB，主力训练） |
| 镜像 | **公共镜像 → 人工智能平台 → CANN 9.0 + PyTorch 2.4.0** |
| 系统盘 | 40GB SSD |
| 公网 IP | 现在购买（1M 带宽，按流量计费） |
| 登录凭证 | 密钥对 `ascend-key` |
| 名称 | `thruster-ascend` |

> 或者选 `ai1s.large.1`（8GB，0.3元/小时）做推理和 SHAP，`ai2.xlarge.1`（32GB，8元/小时）做训练。

### 4.2 修改设备配置

标准 PyTorch 代码，无自定义 CUDA 算子。只需改 **3 个文件**，每个文件 1-2 行：

#### 文件 1：`model_train_v2.py`


#### 文件 2：`model_test_v2.py`

```python
# 文件头部新增：
import torch_npu

# 修改设备检测（第 46 行附近）：
device = torch.device('npu:0' if torch.npu.is_available() else 'cpu')
```

#### 文件 3：`model_shap_v2.py`

```python
# 文件头部新增：
import torch_npu
```

## 八、常见问题

| 问题 | 解决方法 |
|:-----|:---------|
| `npu-smi info` 显示无设备 | 确认选的是鲲鹏架构+昇腾规格，不是 x86 |
| `import torch_npu` 报错 | 确认使用官方 CANN+PyTorch 镜像，不要自己装 |
| 训练比 CUDA 慢 | 正常，昇腾不需要和 CUDA 比速度，比**结果一致性** |
| 显存不足 | 减小 batch_size 到 32 或 16 |
| 连不上服务器 | 检查安全组是否开放 22 端口 |
